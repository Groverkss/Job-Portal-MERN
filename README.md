# Job Portal

Job Portal built using MERN stack as part of DASS course at IIIT-H

## Installing

### Backend

Install modules required for backend REST API

```bash

cd backend/
npm install

```

Fill details required in `sample.env` and change its name to `.env`

```bash

mv sample.env .env

```

Run backend code

```bash

npm start

```

### Frontend

Install required modules for frontend

```bash

cd frontend/
npm install

```

Run frontend code

```bash

npm start

```
